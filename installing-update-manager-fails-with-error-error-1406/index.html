Installing Update Manager fails with error: error 1406

安装Update Manager时报错

错误如下：

这是因为之前安装卸载不干净导致的：

解决办法：

运行 -> regedit (or regedt32 )-> HKEY_LOCAL_MACHINE -> SOFTWARE -> Wow6432Node -> VMware, Inc.(删除此项)

-> 点击重试
