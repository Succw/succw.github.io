wordpress固定链接

WordPress 默认使用带有很多问号和数字的 URL，但 WordPress 允许自定义链接形式，以提高美感、可用性和向前兼容性。您可阅读参考资料以了解更多。

个人配置：

http://www.succw.com/%category%/%postname% 在Nginx.conf中添加 location / {

try_files $uri $uri/ /index.php?p=$request_uri;

}
